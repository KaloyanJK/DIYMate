Quick Guide

How To Use DIYMate

Follow these steps to go from a rough idea to a complete DIY plan with AI help.

1. Create an account

  • Open Register and complete your details.
  • Log in using username or email.
  • Use Remember me if you want a longer session.

2. Create your first project

  • Go to Projects and click New Project.
  • Add title, description, dimensions, and budget.
  • Save to open the project detail page.

3. Generate your AI plan

  • Click Generate AI Plan on the project page.
  • Review materials, safety, and step-by-step instructions.
  • The plan is cached so you can reopen it quickly.

4. Add inspirations and drawing

  • Generate inspiration ideas for style direction.
  • Premium users can generate one technical drawing per project.
  • Save or download drawing outputs from the same page.
